🏢 App RH - Sistema CRUD de Gestão de Vagas e Candidatos
⚠️ Projeto de Estudo - Desenvolvido para fins educacionais

📋 Sobre o Projeto
Este é um sistema CRUD completo (Create, Read, Update, Delete) desenvolvido durante meus estudos de Java e Spring Boot. A aplicação gerencia vagas de emprego e candidatos, implementando todas as operações básicas de um sistema web.

🎯 CRUD - Funcionalidades Implementadas
✅ CREATE (Criar)
Vagas: Cadastro de novas vagas de emprego

Candidatos: Inscrição de candidatos nas vagas

Formulários com validação de dados

✅ READ (Ler)
Listagem: Visualização de todas as vagas cadastradas

Detalhes: Visualização completa de cada vaga e seus candidatos

Busca: Recuperação de dados específicos do banco

✅ UPDATE (Atualizar)
Edição: Modificação de informações das vagas

Atualização: Persistência das alterações no banco

✅ DELETE (Excluir)
Vagas: Remoção de vagas do sistema

Candidatos: Exclusão de candidatos específicos

Operações com confirmação

🛠️ Tecnologias Utilizadas no CRUD
Backend - Stack Spring
Spring Boot 3.4.2 - Framework principal

Spring Data JPA - Persistência e repositórios

Spring MVC - Arquitetura web MVC

Hibernate - ORM para mapeamento objeto-relacional

MySQL - Banco de dados relacional

Thymeleaf - Template engine para views

Frontend
Bootstrap 5 - Interface responsiva

HTML5 + Thymeleaf - Templates dinâmicos

JavaScript - Interatividade básica
